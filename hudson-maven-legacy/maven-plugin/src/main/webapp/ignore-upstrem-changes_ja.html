チェックすると、このプロジェクトのPOMを解析して、依存するスナップショットがこのHudson上でビルドされるかどうかチェックします。 もしビルドされるなら、依存するジョブがビルドされスナップショットのjarファイルが作成されるときはいつでも、 このプロジェクトのビルドをスケジュールするように、Hudsonは依存性を設定します。

この機能は、自動的に継続インテグレーションを実行するのに便利です。Hudsonは、POMの<plugin>や<extension>と同様に、 <dependency>からもスナップショットの依存性をチェックします。

もし動作に問題があるようでしたら、このオプションのチェックをはずしてください。
